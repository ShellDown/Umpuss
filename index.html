Comment faire jouer de la musique a Umpus ?

Il vous suffit de faire la commande suivante : d!play "le nom de votre musique", Exemple d!play L' Etoile sur le maillot.

Comment voir les commandes disponible ?

Pour voir toutes les commandes disponible il vous suffit juste de faire d!help.

d!suggest c'est quoi ?

La commande d!suggest est une commande qui vous permez de nous faires parvenir vos suggestions pour améliorer le bot.
